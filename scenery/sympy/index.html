NAGINI://SYMPY computer algebra in the worker :: sympy + matplotlib + strictyaml
idle, boots on first run

Real computer algebra in the browser: every editor below runs SymPy inside the Pyodide worker, with matplotlib for the figures, input() feeding sympify, and strictyaml installed from PyPI through micropip. One shared manager is created on your first run and reused everywhere. It uses the snapshotCache option, so after the first visit the interpreter restores from IndexedDB in about 100 ms on reload; the sympy and matplotlib downloads still run each time, which is what you wait for on the first run of a fresh tab (30 to 90 seconds cold, much less after).
