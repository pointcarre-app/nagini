Service worker indisponible : servez cette page via http://localhost:8010 (contexte sécurisé requis), puis rechargez.

Comment ça marche. Une app FastAPI complète (routes, validation pydantic, OpenAPI) tourne dans Pyodide, dans cet onglet. Un service worker intercepte les fetch() de la page vers ./api/* et les livre à l'app Python, qui répond. Ouvrez l'onglet réseau des DevTools : les requêtes sont réelles, servies par « ServiceWorker », avec leurs statuts 200, 404, 422.

Limite assumée : rien n'écoute sur un port. Un autre onglet, un curl ou une autre machine ne peuvent pas appeler cette API ; c'est l'illusion parfaite, mais locale. Pour exposer pour de vrai : uvicorn et un déploiement.
